# Duplicate Finder

This project helps in finding duplicate files in a specified directory. It scans the files and alerts the user about any duplicates it encounters.

## Installation

To install the project, you can clone it from GitHub:

```bash
git clone https://github.com/JB007CZPRG/DuplicateFinder.git
```

## Usage

To use the Duplicate Finder, navigate to the project directory and run:

```bash
python find_duplicates.py /path/to/directory
```

Replace `/path/to/directory` with the path of the directory you want to scan.

## Features
- Fast and efficient scanning
- Easy to use
- Supports large directories
